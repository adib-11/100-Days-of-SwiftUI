# 100 Days of SwiftUI 📱

A public log of my journey mastering iOS development through Paul Hudson's [100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui) curriculum. 

As an engineering student, I am using this repository not just to follow tutorials, but to build a strong foundation in application architecture, state management, and modern UI design using Swift.

## 🛠 Technologies & Frameworks
* **Language:** Swift
* **UI Framework:** SwiftUI
* **Machine Learning:** CoreML & Create ML (BetterRest)
* **Version Control:** Git & GitHub
